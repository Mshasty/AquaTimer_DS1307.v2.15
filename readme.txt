Дисплей с интерфейсом i2c вешается параллельно с RTC DS1307 и ноги D4 - D9 освобождаются для
Ваших фантазий ))

uint8_t BLset[2] = {255, 55}; // 255 - яркость в дневном режиме, 55 - яркость в ночном режиме
                              // (теперь запоминается в энергонезависимой памяти)

Настройка подсветки производится в подменю "Set backlight" системного меню. 

Для регулировки подсветки на дисплее с интерфейсом i2c, подключите D10, как показано на рисун-
ке LCD_i2c_backlight.png, желательно, через резистор 220 Ом.

Кнопки в рабочем режиме:
  left -  вкл/выкл 1-го канала (свет);
  right - вкл/выкл 2-го канала (помпа);
  up -    смена режима подсветки экрана;
  down -  немедленная подача корма.
